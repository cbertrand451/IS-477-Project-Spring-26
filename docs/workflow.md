# Reproducible Workflow Instructions
This document describes all steps required to fully reproduce the project workflow from raw data acquisition to the end analysis.

The workflow can be executed in two ways:

1. Using the run_all.py script
2. Running each script manually in order

Both methods regenerate the entire project from scratch using the scripts stored in the scripts directory.

---

# Prerequisites

To repeat the workflow, the following software must be installed:

### Required:
- Python 3.10+
- Required Python packages (requirements.txt)
- Internet access for the scraping scripts (to access basketball-reference.com)

### Installing requirements:
Inside your project environment:

pip install -r requirements.txt

---

# Workflow Overview

The workflow consists of the following automated stages:

Data Acquisition

Data Cleaning

Data Integration

Analysis & Visualization

Each stage corresponds to a script in the scripts/ directory.

---

# Running the Workflow with run_all.py

The entire workflow can be reproduced with a single python script.

The user should run:

python run_all.py

This script sequentially executes:

- acquire_draft_data.py
- acquire_rookie_data.py
- chicago_bulls_season_data_sha256.py
- check_sha256.py
- clean_draft_data.py
- clean_bulls_data.py
- integrate_cleaned_datasets.py
- clean_integrated_data.py
- dataset_eda.py
- modeling.py

The rookie data scraping script may take 2-5 minutes because a timer is used between requests. This prevents sending too many requests to basketball-reference.com too quickly.

---

# Running the Workflow Manually

Each script can also be run manually from the project root.

### Data acquisition

python scripts/acquire/acquire_draft_data.py

python scripts/acquire/acquire_rookie_data.py

python scripts/acquire/chicago_bulls_season_data_sha256.py

python scripts/acquire/check_sha256.py

### Data cleaning

python scripts/clean/clean_draft_data.py

python scripts/clean/clean_bulls_data.py

### Data integration

python scripts/integrate/integrate_cleaned_datasets.py

### Final cleaning

python scripts/clean/clean_integrated_data.py

### Analysis and modeling

python scripts/analyze/dataset_eda.py

python scripts/analyze/modeling.py

# Required Input Files

The workflow requires one manually downloaded dataset:

data/raw/bulls_data.csv

The draft data and rookie year stat files are acquired programmatically from basketball-reference.com.

The manually downloaded Bulls season dataset also needs a checksum file generated by the workflow:

data/raw/bulls_data.sha256
